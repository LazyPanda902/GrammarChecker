\# Grammar App



A simple grammar and rewriting web app built with React and Node.js.



\## Features



\- Fix grammar and spelling

\- Rewrite text clearly

\- Convert writing to formal tone

\- Convert writing to casual tone

\- Uses a backend API layer so the API key stays private



\## Tech Stack



\- Frontend: React + Vite

\- Backend: Node.js + Express

\- AI: Google Gemini API
